Syntax:

Logic 1: 1[length]
Logic 0: 0[length]
Data: D[bytes]
Color: C<Color name>
Seperate fields by commas, ideally without spaces.
Example: 125,025,Cred,1100,0500,Cgreen,D7,Cblue,150
Set the default bit length in the boxes below, as well as the desired SVG graph dimensions.
The bit length is required to create the byte entries (Bytes*8*length).
DO NOT INCLUDE OTHER PUNCTUATION OR LETTERS IN ANY FIELDS OR SOMETHING WILL BREAK.
